# Security Scanner

**Cencori Scan** is a security scanner for AI apps that detects hardcoded secrets, PII leaks, exposed routes, and security vulnerabilities — with AI-powered auto-fix.

## Quick Start

```bash
npx @cencori/scan
```

That's it. Run it in any project directory to instantly scan for security issues.

## What It Detects

### Secrets & API Keys

Detects hardcoded API keys from 20+ providers:
- OpenAI (`sk-...`, `sk-proj-...`)
- Anthropic (`sk-ant-...`)
- Google AI (`AIza...`)
- Supabase, Stripe, AWS, GitHub
- Firebase, SendGrid, Twilio, and more

### Personal Information (PII)

Identifies sensitive personal data in your code:
- Email addresses
- Phone numbers (US & International)
- Social Security Numbers
- Credit card numbers

### Security Vulnerabilities

Finds common security issues:
- SQL injection patterns
- XSS vulnerabilities (`innerHTML`, `dangerouslySetInnerHTML`)
- Insecure CORS configuration
- Hardcoded passwords
- Debug modes in production
- Eval usage and code execution risks

### Exposed Routes

Detects unprotected API routes:
- Next.js API routes without authentication
- Express routes without auth middleware
- Dashboard/admin routes without protection

## Security Scoring

Every scan provides an **A through F tier grading**:
