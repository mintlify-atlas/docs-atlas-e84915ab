## Basic Usage

The `chat()` method sends a non-streaming chat completion request:

```python
from cencori import Cencori

cencori = Cencori(api_key="csk_...")

response = cencori.ai.chat(
    messages=[{"role": "user", "content": "What is AI?"}],
    model="gpt-4o"
)

print(response.content)
```

## Message Format

Messages have two required fields: `role` and `content`.

```python
messages = [
    {"role": "system", "content": "You are a helpful coding assistant."},
    {"role": "user", "content": "How do I reverse a string in Python?"},
    {"role": "assistant", "content": "You can use slicing: text[::-1]"},
    {"role": "user", "content": "Can you show a full example?"}
]

response = cencori.ai.chat(messages=messages)
```

### Roles

- `system`: Sets the behavior and context for the assistant
- `user`: User messages or prompts
- `assistant`: Previous assistant responses (for conversation history)

## Parameters

### Temperature

Controls randomness (0-1). Lower values are more deterministic:

```python
# More creative (default varies by model)
response = cencori.ai.chat(
    messages=[{"role": "user", "content": "Write a creative story"}],
    temperature=0.9
)

# More focused and deterministic
response = cencori.ai.chat(
    messages=[{"role": "user", "content": "What is 2+2?"}],
    temperature=0.1
)
```

### Max Tokens

Limits the response length:

```python
response = cencori.ai.chat(
    messages=[{"role": "user", "content": "Explain quantum computing"}],
    max_tokens=100  # Keep response concise
)
```

### User ID

Track usage per user for rate limiting:

```python
response = cencori.ai.chat(
    messages=[{"role": "user", "content": "Hello"}],
    user_id="user_123"  # Optional user identifier
)
```

## Complete Example

```python
from cencori import Cencori

cencori = Cencori(api_key="csk_...")

response = cencori.ai.chat(
    messages=[
        {"role": "system", "content": "You are a Python expert."},
        {"role": "user", "content": "Explain list comprehensions"}
    ],
    model="gpt-4o",
    temperature=0.7,
    max_tokens=500,
    user_id="user_456"
)

print(f"Response: {response.content}")
print(f"Model: {response.model}")
print(f"Provider: {response.provider}")
print(f"Cost: ${response.cost_usd:.6f}")
print(f"Tokens: {response.usage.total_tokens}")
```

## Text Completions

For simple text completions without message structure:

```python
response = cencori.ai.completions(
    prompt="Write a haiku about coding",
    model="gpt-4o",
    temperature=0.8
)

print(response.content)
```

The `completions()` method is a convenience wrapper that converts the prompt to a user message internally.

## Embeddings

Generate vector embeddings for semantic search and RAG:

```python
# Single text
response = cencori.ai.embeddings(
    input="Hello world",
    model="text-embedding-3-small"
)

print(f"Embedding dimension: {len(response.embeddings[0])}")
print(f"First few values: {response.embeddings[0][:5]}")

# Multiple texts
response = cencori.ai.embeddings(
    input=["Hello world", "Goodbye world", "AI is amazing"],
    model="text-embedding-3-small"
)

print(f"Generated {len(response.embeddings)} embeddings")
for i, embedding in enumerate(response.embeddings):
    print(f"Text {i+1}: {len(embedding)} dimensions")
```

## Async Support

All methods have async counterparts for concurrent operations:

```python
import asyncio
from cencori import Cencori

async def main():
    cencori = Cencori(api_key="csk_...")
    
    # Async chat
    response = await cencori.ai.async_chat(
        messages=[{"role": "user", "content": "Hello!"}],
        model="gpt-4o"
    )
    print(response.content)
    
    # Async completions
    response = await cencori.ai.async_completions(
        prompt="Write a poem",
        model="claude-3-haiku"
    )
    print(response.content)
    
    # Async embeddings
    response = await cencori.ai.async_embeddings(
        input="AI infrastructure"
    )
    print(len(response.embeddings[0]))

asyncio.run(main())
```

## Response Metadata

Access detailed response information:

```python
response = cencori.ai.chat(
    messages=[{"role": "user", "content": "Hello"}]
)

# Token usage
print(f"Prompt tokens: {response.usage.prompt_tokens}")
print(f"Completion tokens: {response.usage.completion_tokens}")
print(f"Total tokens: {response.usage.total_tokens}")

# Cost tracking
print(f"Cost: ${response.cost_usd:.6f}")

# Model info
print(f"Model: {response.model}")
print(f"Provider: {response.provider}")

# Finish reason
print(f"Finish reason: {response.finish_reason}")  # stop, length, content_filter, error
```

## Best Practices

### System Prompts

Use system messages to set consistent behavior:

```python
system_prompt = """You are a helpful customer service assistant.
Always be polite and professional.
If you don't know something, admit it."""

response = cencori.ai.chat(
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": "I need help with my order"}
    ]
)
```

### Conversation Management

Manage conversation history efficiently:

```python
class ChatSession:
    def __init__(self, cencori, system_prompt=None):
        self.cencori = cencori
        self.messages = []
        if system_prompt:
            self.messages.append({"role": "system", "content": system_prompt})
    
    def send(self, user_message, **kwargs):
        self.messages.append({"role": "user", "content": user_message})
        response = self.cencori.ai.chat(messages=self.messages, **kwargs)
        self.messages.append({"role": "assistant", "content": response.content})
        return response

# Usage
session = ChatSession(cencori, system_prompt="You are a coding tutor.")
response1 = session.send("What is a function?")
response2 = session.send("Can you show an example?")  # Remembers context
```

### Token Management

Monitor and limit token usage:

```python
MAX_TOKENS_PER_REQUEST = 1000

response = cencori.ai.chat(
    messages=messages,
    max_tokens=MAX_TOKENS_PER_REQUEST
)

if response.finish_reason == "length":
    print("Response was truncated due to token limit")
```
