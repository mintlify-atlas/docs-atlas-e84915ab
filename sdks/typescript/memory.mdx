# Memory & RAG

The Memory API provides vector storage for building RAG (Retrieval-Augmented Generation) applications, storing conversation history, and implementing semantic search.

## Overview

Cencori Memory is a managed vector database that:

- **Automatic embeddings**: Content is embedded automatically
- **Semantic search**: Find relevant content by meaning, not keywords
- **Namespaces**: Organize memories by project, user, or purpose
- **RAG integration**: Built-in chat with automatic context retrieval
- **TTL support**: Automatic expiration for temporary data

## Quick Start

```typescript
import { Cencori } from 'cencori';

const cencori = new Cencori({ apiKey: process.env.CENCORI_API_KEY });

// Store a memory
await cencori.memory.store({
  namespace: 'product-docs',
  content: 'Cencori provides AI Gateway, Compute, Workflow, and Storage.',
  metadata: { source: 'overview.md', section: 'features' }
});

// Search memories
const results = await cencori.memory.search({
  namespace: 'product-docs',
  query: 'What does Cencori provide?',
  limit: 5
});

console.log(results.results[0].content);
```

See `packages/sdk/src/memory/index.ts:67` for the `MemoryClient` implementation.

## Searching Memories

### Semantic Search

Find relevant memories by meaning:


### Search Result Format

```typescript
interface SearchResult {
  results: Memory[];     // Matching memories
  query: string;         // Original query
  namespace: string;     // Namespace searched
  count: number;         // Number of results
  latencyMs: number;     // Search duration
}

interface Memory {
  id: string;                         // Memory ID
  namespace: string;                  // Namespace
  content: string;                    // Stored content
  metadata: Record<string, unknown>;  // Custom metadata
  similarity?: number;                // Similarity score (0-1)

  createdAt: string;                  // Creation time
  updatedAt?: string;                 // Last update
}
```

## RAG (Retrieval-Augmented Generation)

### Basic RAG

Chat with automatic context retrieval:

```typescript
const response = await cencori.ai.rag({
  model: 'gpt-4o',
  messages: [
    { role: 'user', content: 'What are the main features of Cencori?' }
  ],
  namespace: 'product-docs',
  limit: 5,         // Retrieve top 5 relevant memories
  threshold: 0.5    // Only use memories with similarity >= 0.5
});

console.log('Response:', response.message.content);
console.log('Sources used:', response.sources?.length);

// View retrieved sources
response.sources?.forEach(source => {
  console.log(`[${source.similarity.toFixed(2)}] ${source.content.slice(0, 100)}...`);
  console.log('Metadata:', source.metadata);
});
```

See `packages/sdk/src/ai/index.ts:441` for the `rag()` method.

### RAG Streaming

Stream responses with context retrieval:

```typescript
for await (const chunk of cencori.ai.ragStream({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Explain the pricing model' }],
  namespace: 'product-docs',
  limit: 5
})) {
  if (chunk.type === 'sources') {
    // Received retrieved sources
    console.log('\nRetrieved sources:');
    chunk.sources?.forEach((source, i) => {
      console.log(`${i + 1}. [${source.similarity.toFixed(2)}] ${source.content.slice(0, 80)}...`);
    });
    console.log();
  } else if (chunk.type === 'content') {
    // Streaming response
    process.stdout.write(chunk.delta);
  }
}
```

See `packages/sdk/src/ai/index.ts:499` for the `ragStream()` method.

### Include/Exclude Sources

```typescript
const response = await cencori.ai.rag({
  model: 'gpt-4o',
  messages,
  namespace: 'docs',
  includeSources: false  // Don't return sources in response (faster)
});

// Response only includes the AI's answer
console.log(response.message.content);
```

## Common Use Cases

### Conversation History

Store and retrieve chat context:

```typescript
import { ChatMessage } from 'cencori';

const conversationHistory: ChatMessage[] = [];

async function chat(userMessage: string, userId: string) {
  // Store user message
  await cencori.memory.store({
    namespace: `chat-${userId}`,
    content: userMessage,
    metadata: {
      role: 'user',
      timestamp: new Date().toISOString()
    }
  });

  // Search relevant history
  const context = await cencori.memory.search({
    namespace: `chat-${userId}`,
    query: userMessage,
    limit: 5
  });

  // Build context from history
  const contextMessages: ChatMessage[] = context.results.map(m => ({
    role: m.metadata.role as 'user' | 'assistant',
    content: m.content
  }));

  // Chat with context
  const response = await cencori.ai.chat({
    model: 'gpt-4o',
    messages: [
      ...contextMessages,
      { role: 'user', content: userMessage }
    ]
  });

  // Store assistant response
  await cencori.memory.store({
    namespace: `chat-${userId}`,
    content: response.content,
    metadata: {
      role: 'assistant',
      timestamp: new Date().toISOString()
    }
  });

  return response;
}
```

### Document Q&A

### Semantic Search API

Build a search endpoint:

```typescript
import express from 'express';
import { Cencori } from 'cencori';

const app = express();
const cencori = new Cencori({ apiKey: process.env.CENCORI_API_KEY });

app.post('/api/search', async (req, res) => {
  const { query, namespace = 'default', limit = 10 } = req.body;

  try {
    const results = await cencori.memory.search({
      namespace,
      query,
      limit,
      threshold: 0.5
    });

    res.json({
      results: results.results.map(r => ({
        content: r.content,
        metadata: r.metadata,
        similarity: r.similarity
      })),
      count: results.count,
      latencyMs: results.latencyMs
    });
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});

app.listen(3000);
```

## Best Practices

### Chunk Size

Keep chunks focused (300-500 tokens):

```typescript
function chunkText(text: string, chunkSize = 500): string[] {
  const sentences = text.split(/[.!?]+/);
  const chunks: string[] = [];
  let current = '';

  for (const sentence of sentences) {
    if ((current + sentence).length > chunkSize && current) {
      chunks.push(current.trim());
      current = sentence;
    } else {
      current += sentence + '.';
    }
  }

  if (current) chunks.push(current.trim());
  return chunks;
}
```

## Complete Example

Production-ready RAG implementation:

```typescript
import { Cencori } from 'cencori';

class DocumentRAG {
  private cencori: Cencori;
  private namespace: string;

  constructor(namespace: string) {
    this.cencori = new Cencori({ apiKey: process.env.CENCORI_API_KEY });
    this.namespace = namespace;
  }

  async indexDocument(title: string, content: string) {
    const chunks = this.chunkText(content);

    const memories = await this.cencori.memory.storeBatch(
      this.namespace,
      chunks.map((chunk, i) => ({
        content: chunk,
        metadata: {
          title,
          chunkIndex: i,
          totalChunks: chunks.length,
          indexedAt: new Date().toISOString()
        }
      }))
    );

    return memories;
  }

  async query(question: string, options: { streaming?: boolean } = {}) {
    if (options.streaming) {
      return this.cencori.ai.ragStream({
        model: 'gpt-4o',
        messages: [{ role: 'user', content: question }],
        namespace: this.namespace,
        limit: 5,
        threshold: 0.6
      });
    }

    return this.cencori.ai.rag({
      model: 'gpt-4o',
      messages: [{ role: 'user', content: question }],
      namespace: this.namespace,
      limit: 5,
      threshold: 0.6
    });
  }

  async search(query: string) {
    return this.cencori.memory.search({
      namespace: this.namespace,
      query,
      limit: 10,
      threshold: 0.5
    });
  }

  private chunkText(text: string, maxSize = 500): string[] {
    // Implement chunking logic
    return text.match(new RegExp(`.{1,${maxSize}}`, 'g')) || [];
  }
}

// Usage
const rag = new DocumentRAG('company-docs');

await rag.indexDocument('Overview', 'Cencori is...');
const answer = await rag.query('What is Cencori?');
console.log(answer.message.content);
```
