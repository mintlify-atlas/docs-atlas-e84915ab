## Core Features

All Cencori SDKs share these capabilities:

### Unified API Surface

One SDK for all AI infrastructure primitives:

```typescript
import { Cencori } from 'cencori';

const cencori = new Cencori({ apiKey: 'csk_...' });

// AI Gateway
await cencori.ai.chat({ model: 'gpt-4o', messages });

// Memory (RAG)
await cencori.memory.search({ namespace: 'docs', query: '...' });

// Compute (coming soon)
await cencori.compute.run('function-name', { input });

// Workflow (coming soon)
await cencori.workflow.trigger('pipeline-id', { data });
```

### Built-in Security

- **API Key Authentication**: Secure access with `csk_` prefixed keys
- **Rate Limiting**: Automatic rate limit handling with `RateLimitError`
- **Content Safety**: Built-in safety checks with detailed violation reasons
- **Error Handling**: Type-safe error classes for every failure mode

### Automatic Observability

- **Request Logging**: Every API call is logged to your dashboard
- **Token Tracking**: Automatic usage tracking for cost optimization
- **Latency Metrics**: Built-in performance monitoring
- **Error Reporting**: Detailed error traces for debugging

### Production Ready

- **Automatic Retries**: Exponential backoff for transient failures
- **Streaming Support**: First-class streaming for chat and completions
- **Type Safety**: Full TypeScript definitions and Python type hints
- **Framework Integrations**: Native support for Vercel AI SDK, TanStack AI, LangChain

## Authentication

All SDKs support multiple authentication methods:

<Tabs>
  <Tab title="Environment Variable">
    ```bash
    export CENCORI_API_KEY=csk_...
    ```
    
    The SDK will automatically read from `CENCORI_API_KEY`.
  </Tab>
  
  <Tab title="Constructor">
    ```typescript
    import { Cencori } from 'cencori';
    
    const cencori = new Cencori({ 
      apiKey: 'csk_...' 
    });
    ```
  </Tab>
  
  <Tab title="Custom Headers">
    ```typescript
    const cencori = new Cencori({ 
      apiKey: 'csk_...',
      headers: {
        'X-Custom-Header': 'value'
      }
    });
    ```
  </Tab>
</Tabs>
