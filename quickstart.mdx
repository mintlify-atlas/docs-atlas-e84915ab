Get started with Cencori and make your first AI request in minutes.

## Prerequisites

- A Cencori account (sign up at [cencori.com](https://cencori.com))
- Node.js 18+, Python 3.8+, or Go 1.19+ installed

## Steps

<Steps>
  <Step title="Install the SDK">
    Choose your preferred language and install the Cencori SDK:

    <CodeGroup>
    ```bash npm
    npm install cencori
    ```

    ```bash yarn
    yarn add cencori
    ```

    ```bash pnpm
    pnpm add cencori
    ```

    ```bash pip
    pip install cencori
    ```

    ```bash go
    go get github.com/cencori/cencori-go
    ```
    </CodeGroup>
  </Step>

  <Step title="Get your API key">
    1. Sign in to your [Cencori dashboard](https://cencori.com/dashboard)
    2. Create a new project
    3. Navigate to **API Keys**
    4. Click **Create API Key**
    5. Copy your secret key (starts with `csk_`)

    <Warning>
      Store your API key securely. Never commit it to version control.
    </Warning>
  </Step>

  <Step title="Set up your environment">
    Add your API key to your environment variables:

    ```bash
    export CENCORI_API_KEY=csk_your_secret_key
    ```

    Or create a `.env` file:

    ```bash
    CENCORI_API_KEY=csk_your_secret_key
    ```
  </Step>

  <Step title="Make your first request">
    <Tabs>
      <Tab title="TypeScript">
        ```typescript
        import { Cencori } from 'cencori';

        const cencori = new Cencori({
          apiKey: process.env.CENCORI_API_KEY
        });

        const response = await cencori.ai.chat({
          messages: [{ role: 'user', content: 'Hello!' }],
          model: 'gpt-4o'
        });

        console.log(response.content);
        // Output: Hello! How can I help you today?
        ```
      </Tab>

      <Tab title="Python">
        ```python
        from cencori import Cencori
        import os

        cencori = Cencori(api_key=os.getenv("CENCORI_API_KEY"))

        response = cencori.ai.chat(
            messages=[{"role": "user", "content": "Hello!"}],
            model="gpt-4o"
        )

        print(response.content)
        # Output: Hello! How can I help you today?
        ```
      </Tab>

      <Tab title="Go">
        ```go
        package main

        import (
            "context"
            "fmt"
            "os"
            "github.com/cencori/cencori-go"
        )

        func main() {
            client, err := cencori.NewClient(
                cencori.WithAPIKey(os.Getenv("CENCORI_API_KEY")),
            )
            if err != nil {
                panic(err)
            }

            resp, err := client.Chat.Create(context.Background(), &cencori.ChatParams{
                Model: "gpt-4o",
                Messages: []cencori.Message{
                    {Role: "user", Content: "Hello!"},
                },
            })
            if err != nil {
                panic(err)
            }

            fmt.Println(resp.Choices[0].Message.Content)
            // Output: Hello! How can I help you today?
        }
        ```
      </Tab>

      <Tab title="cURL">
        ```bash
        curl https://api.cencori.com/v1/ai/chat \
          -H "CENCORI_API_KEY: $CENCORI_API_KEY" \
          -H "Content-Type: application/json" \
          -d '{
            "model": "gpt-4o",
            "messages": [{"role": "user", "content": "Hello!"}]
          }'
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Try streaming">
    Get real-time responses with streaming:

    <Tabs>
      <Tab title="TypeScript">
        ```typescript
        const stream = cencori.ai.chatStream({
          messages: [{ role: 'user', content: 'Tell me a story' }],
          model: 'gpt-4o'
        });

        for await (const chunk of stream) {
          process.stdout.write(chunk.delta);
        }
        ```
      </Tab>

      <Tab title="Python">
        ```python
        for chunk in cencori.ai.chat_stream(
            messages=[{"role": "user", "content": "Tell me a story"}],
            model="gpt-4o"
        ):
            print(chunk.delta, end="", flush=True)
        ```
      </Tab>

      <Tab title="Go">
        ```go
        stream, err := client.Chat.Stream(ctx, &cencori.ChatParams{
            Model: "gpt-4o",
            Messages: []cencori.Message{
                {Role: "user", Content: "Tell me a story"},
            },
        })

        for chunk := range stream {
            if len(chunk.Choices) > 0 {
                fmt.Print(chunk.Choices[0].Delta.Content)
            }
        }
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>

## Common use cases

### Chat completions

Build conversational AI with multi-turn context:

```typescript
const response = await cencori.ai.chat({
  messages: [
    { role: 'system', content: 'You are a helpful assistant.' },
    { role: 'user', content: 'What is Cencori?' }
  ],
  model: 'gpt-4o',
  temperature: 0.7
});
```

### Switch providers

Change AI providers with one parameter:

```typescript
// Use OpenAI
const openai = await cencori.ai.chat({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello' }]
});

// Use Anthropic
const anthropic = await cencori.ai.chat({
  model: 'claude-3-opus',
  messages: [{ role: 'user', content: 'Hello' }]
});

// Use Google Gemini
const gemini = await cencori.ai.chat({
  model: 'gemini-2.5-flash',
  messages: [{ role: 'user', content: 'Hello' }]
});
```

### RAG with memory

Store and retrieve context for smarter responses:

```typescript
// Store a memory
await cencori.memory.store({
  namespace: 'docs',
  content: 'Cencori is an AI infrastructure platform',
  metadata: { source: 'documentation' }
});

// RAG with automatic context retrieval
const response = await cencori.ai.rag({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'What is Cencori?' }],
  namespace: 'docs'
});
```
