## Endpoint

```
POST https://api.cencori.com/api/ai/images/generate
```

Generate images from text prompts using DALL-E, GPT Image, Gemini, or Imagen models.

## Authentication

Requires API key in `Authorization` header or `CENCORI_API_KEY` header.

## Request Body

<ParamField body="prompt" type="string" required>
  Text description of the desired image. Be specific and detailed for best results.
  
  Maximum length: 4000 characters
</ParamField>

<ParamField body="model" type="string">
  Image generation model to use.
  
  Default: `dall-e-3`
  
  Available models:
  - `gpt-image-1.5` - Best text rendering, top ELO rating
  - `gpt-image-1` - ChatGPT image generation  
  - `dall-e-3` - High quality, creative
  - `dall-e-2` - Fast, cost-effective
  - `gemini-3-pro-image` - High photorealism, fast
  - `nano-banana-pro` - Alias for Gemini 3 Pro
  - `gemini-3.1-flash-image` - Reasoning-guided, up to 4K
  - `nano-banana-2` - Alias for Gemini 3.1 Flash
  - `imagen-3` - Google Imagen 3
</ParamField>

<ParamField body="n" type="number">
  Number of images to generate (1-4).
  
  Default: `1`
  
  Note: GPT Image models only support n=1
</ParamField>

<ParamField body="size" type="string">
  Image dimensions.
  
  Available sizes:
  - `256x256`
  - `512x512`
  - `1024x1024` (default)
  - `1024x1792` (portrait)
  - `1792x1024` (landscape)
  - `1536x1024`
  - `1024x1536`
</ParamField>

<ParamField body="quality" type="string">
  Image quality: `standard` or `hd`
  
  Default: `standard`
  
  Note: Only supported by DALL-E 3
</ParamField>

<ParamField body="style" type="string">
  Image style: `vivid` or `natural`
  
  Default: `vivid`
  
  Note: Only supported by DALL-E 3
</ParamField>

<ParamField body="responseFormat" type="string">
  Response format: `url` or `b64_json`
  
  Default: `url`
  
  Note: GPT Image models always return base64
</ParamField>

## Response

<ResponseField name="images" type="array">
  Array of generated images
  
  <Expandable title="Image Object">
    <ResponseField name="url" type="string">
      Image URL (if responseFormat is 'url')
    </ResponseField>
    <ResponseField name="b64_json" type="string">
      Base64-encoded image data (if responseFormat is 'b64_json')
    </ResponseField>
    <ResponseField name="revisedPrompt" type="string">
      Revised prompt used for generation (DALL-E 3 only)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="model" type="string">
  Model used for generation
</ResponseField>

<ResponseField name="provider" type="string">
  Provider used: `openai` or `google`
</ResponseField>

### Using OpenAI SDK

```typescript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: process.env.CENCORI_API_KEY,
  baseURL: 'https://api.cencori.com/v1'
});

const response = await client.images.generate({
  model: 'dall-e-3',
  prompt: 'A white siamese cat',
  n: 1,
  size: '1024x1024'
});

const imageUrl = response.data[0].url;
console.log(imageUrl);
```

### Download and Save Image

```typescript
import fs from 'fs';

const response = await client.images.generate({
  model: 'dall-e-3',
  prompt: 'A serene Japanese garden',
  response_format: 'b64_json'
});

const base64Data = response.data[0].b64_json;
const buffer = Buffer.from(base64Data, 'base64');
fs.writeFileSync('image.png', buffer);
```

## Prompt Engineering Tips

### Be Specific

```
Bad:  "A dog"
Good: "A golden retriever puppy playing in a field of flowers at sunset"
```

### Include Style Details

```
"Digital art", "oil painting", "photorealistic", "watercolor", 
"3D render", "minimalist", "vintage poster", "Studio Ghibli style"
```

### Specify Composition

```
"close-up portrait", "wide angle shot", "aerial view", 
"centered composition", "rule of thirds"
```

### Add Lighting Details

```
"golden hour lighting", "dramatic shadows", "soft diffused light",
"neon lights", "volumetric lighting", "cinematic lighting"
```

### Example Detailed Prompt

```json
}
```

## Best Practices

1. **Choose the right model**
   - Use `gpt-image-1.5` for text-heavy images
   - Use `dall-e-3` for creative, artistic images
   - Use `gemini-3-pro-image` for fast photorealistic results
   - Use `dall-e-2` for quick iterations and testing

2. **Optimize prompts**
   - Be specific and descriptive
   - Include style, lighting, and composition
   - Iterate and refine based on results

3. **Consider costs**
   - Test with cheaper models first
   - Use HD quality only when needed
   - Generate multiple variations with `n` parameter

4. **Handle responses**
   - Download and cache generated images
   - URLs expire after 1 hour
   - Use `b64_json` for immediate storage

5. **Content policy**
   - Follow provider content policies
   - Avoid generating harmful content
   - Respect copyright and trademarks
