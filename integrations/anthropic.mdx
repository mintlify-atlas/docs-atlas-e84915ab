# Anthropic Provider Setup

Add your Anthropic API key to Cencori to access Claude Opus, Claude Sonnet, and other Anthropic models through the AI Gateway.

## Adding Your Anthropic API Key

<Steps>
  <Step title="Get your Anthropic API key">
    If you don't have an Anthropic API key:
    
    1. Go to [console.anthropic.com](https://console.anthropic.com)
    2. Sign in or create an account
    3. Navigate to **API Keys**
    4. Click **Create Key**
    5. Copy the key (it starts with `sk-ant-`)
  </Step>

  <Step title="Add key to Cencori">
    In your [Cencori Dashboard](https://cencori.com/dashboard):
    
    1. Go to **Settings** > **Providers**
    2. Click **Add Provider**
    3. Select **Anthropic**
    4. Paste your Anthropic API key
    5. Click **Save**
  </Step>

  <Step title="Test the connection">
    Cencori will automatically test your API key to ensure it's valid.
  </Step>
</Steps>

## Usage Examples

### Using the SDK

<CodeGroup>
```typescript TypeScript
import { Cencori } from 'cencori';

const cencori = new Cencori({
  apiKey: process.env.CENCORI_API_KEY
});

const response = await cencori.ai.chat({
  model: 'claude-3-opus',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.content);
```

```python Python
from cencori import Cencori

cencori = Cencori(api_key="csk_your_key")

response = cencori.ai.chat(
    model="claude-3-opus",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.content)
```

```bash cURL
curl https://api.cencori.com/v1/chat/completions \
  -H "Authorization: Bearer csk_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-3-opus",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```
</CodeGroup>

### Streaming Responses

<CodeGroup>
```typescript TypeScript
const stream = cencori.ai.chatStream({

```
</CodeGroup>

## Advanced Configuration

### Temperature and Parameters

```typescript
const response = await cencori.ai.chat({
  model: 'claude-3-opus',
  messages: [
    { role: 'system', content: 'You are a helpful assistant' },
    { role: 'user', content: 'Write a creative story' }
  ],
  temperature: 1.0,  // More creative (max for Claude)
  maxTokens: 2048
});
```

### System Messages

Claude uses system messages for instructions:

```typescript
const response = await cencori.ai.chat({
  model: 'claude-3-opus',
  messages: [
    {
      role: 'system',
      content: 'You are an expert Python developer. Write clean, documented code.'
    },
    {
      role: 'user',
      content: 'Create a REST API for a todo app'
    }
  ]
});
```

### Long Context

Claude models support very long context:

- **Claude 3 Opus**: 200,000 tokens (~150,000 words)
- **Claude 3 Sonnet**: 200,000 tokens
- **Claude 3 Haiku**: 200,000 tokens

```typescript
const response = await cencori.ai.chat({
  model: 'claude-3-opus',
  messages: [
    {
      role: 'user',
      content: `Analyze this document: ${largeDocument}`
    }
  ],
  maxTokens: 4096
});
```

## Troubleshooting

### Invalid API Key

If you see authentication errors:

1. Check that your Anthropic key starts with `sk-ant-`
2. Verify the key is active in [Anthropic Console](https://console.anthropic.com/settings/keys)
3. Ensure you've added it to Cencori, not using it directly

### Rate Limits

Anthropic rate limits depend on your plan:

- **Free tier**: Limited to 5 requests per minute
- **Paid tier**: Higher limits based on your usage

View your limits in the [Anthropic Console](https://console.anthropic.com/settings/limits).

### Context Length Errors

If you exceed the context window:

- Reduce the length of your prompt
- Lower the `maxTokens` parameter
- Split large documents into smaller chunks
