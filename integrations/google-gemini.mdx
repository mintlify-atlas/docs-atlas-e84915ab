# Google Gemini Provider Setup

Add your Google API key to Cencori to access Gemini models through the AI Gateway.

## Adding Your Google API Key

<Steps>
  <Step title="Get your Google AI API key">
    If you don't have a Google AI API key:
    
    1. Go to [aistudio.google.com](https://aistudio.google.com/app/apikey)
    2. Sign in with your Google account
    3. Click **Create API Key**
    4. Select or create a Google Cloud project
    5. Copy the API key
  </Step>

  <Step title="Add key to Cencori">
    In your [Cencori Dashboard](https://cencori.com/dashboard):
    
    1. Go to **Settings** > **Providers**
    2. Click **Add Provider**
    3. Select **Google Gemini**
    4. Paste your Google AI API key
    5. Click **Save**
  </Step>

  <Step title="Test the connection">
    Cencori will automatically test your API key to ensure it's valid.
  </Step>
</Steps>

## Usage Examples

### Using the SDK

<CodeGroup>
```typescript TypeScript
import { Cencori } from 'cencori';

const cencori = new Cencori({
  apiKey: process.env.CENCORI_API_KEY
});

const response = await cencori.ai.chat({
  model: 'gemini-2.5-flash',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.content);
```

```python Python
from cencori import Cencori

cencori = Cencori(api_key="csk_your_key")

response = cencori.ai.chat(
    model="gemini-2.5-flash",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.content)
```

```bash cURL
curl https://api.cencori.com/v1/chat/completions \
  -H "Authorization: Bearer csk_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-flash",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```
</CodeGroup>

### Streaming Responses

<CodeGroup>
```typescript TypeScript
const stream = cencori.ai.chatStream({
  model: 'gemini-2.5-flash',
  messages: [{ role: 'user', content: 'Tell me about AI' }]
});

for await (const chunk of stream) {
  process.stdout.write(chunk.delta);
}
```

```python Python
for chunk in cencori.ai.chat_stream(
    model="gemini-2.5-flash",
    messages=[{"role": "user", "content": "Tell me about AI"}]
):
    print(chunk.delta, end="", flush=True)
```
</CodeGroup>

## Advanced Configuration

### Temperature and Parameters

```typescript
const response = await cencori.ai.chat({
  model: 'gemini-2.5-flash',
  messages: [{ role: 'user', content: 'Write a creative story' }],
  temperature: 0.9,  // More creative
  maxTokens: 2048
});
```

### System Instructions

Gemini uses system messages for guidance:

```typescript
const response = await cencori.ai.chat({
  model: 'gemini-2.5-flash',
  messages: [
    {
      role: 'system',
      content: 'You are a helpful coding assistant. Write clean Python code.'
    },
    {
      role: 'user',
      content: 'Create a function to sort a list'
    }
  ]
});
```

### Long Context (1M+ tokens)

Gemini 1.5 Pro supports extremely long context:

```typescript
const response = await cencori.ai.chat({
  model: 'gemini-1.5-pro',
  messages: [
    {
      role: 'user',
      content: `Analyze these documents: ${massiveDataset}`
    }
  ],
  maxTokens: 8192
});
```

## Troubleshooting

### Invalid API Key

If you see authentication errors:

1. Verify your API key is from [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Check that the key is active and not restricted
3. Ensure you've added it to Cencori, not using it directly

### Rate Limits

Google Gemini has generous free tier limits:

- **Free tier**: 15 requests per minute, 1,500 requests per day
- **Paid tier**: 360 requests per minute

View your quota in [Google Cloud Console](https://console.cloud.google.com/iam-admin/quotas).

### Model Not Available

Some models may require:

- Enabling the Gemini API in Google Cloud Console
- Upgrading to a paid Google Cloud account
- Regional availability (some models may not be available in all regions)
